Tema 4

Grundlæggende UI udvikling

Beskrivelse og oplevelse

Dette tema introducerede mig for samspillet mellem design og funktionalitet i brugergrænseflader. Gennem værktøjer som Illustrator, CSS og JavaScript arbejdede vi med SVG, animationer og forms, og fik indblik i praksisnære teknologier som darkmode og CSS-variabler.

Selvom jeg i starten havde svært ved at forstå formålet med temaet, fandt jeg gradvist fodfæste i de konkrete læringsmål og kodeopgaver.

Det udleverede kodeskelet var både en hjælp og en udfordring, da mange begreber var nye. Temaets gakkede ramme gjorde dog hele processen sjov og eksperimenterende.

Læring

I dette tema blev jeg for alvor kastet ud i det visuelle og interaktive lag af brugergrænseflader. Det var første gang, jeg arbejdede med SVG-filer, både som grafik i Illustrator og som kode, og det overraskede mig, hvor fleksibelt det faktisk er. At kunne style og animere SVG direkte med CSS eller manipulere det med JavaScript åbnede en ny verden af muligheder.

JavaScript var nyt for mig, og selvom det ofte var udfordrende, begyndte jeg at forstå, hvordan man kan skabe mere levende og dynamiske interfaces, især i arbejdet med forms.

CSS custom properties og variabler gav mig også en helt ny forståelse af, hvordan man kan gøre design mere fleksibelt og overskueligt. Det gjorde det f.eks. nemmere at arbejde med light og dark mode, som jeg fik meget ud af at eksperimentere med.

Jeg blev også glad for at bruge krydsmetoden til idéudvikling, det hjalp mig til at tænke mere kreativt i de tidlige designfaser. Alt i alt føler jeg mig mere sikker i at skabe brugergrænseflader, der både fungerer og føles godt.
